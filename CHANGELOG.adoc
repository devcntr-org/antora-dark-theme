= Changelog

All notable changes to this project will be documented in this file.
Each release has a short summary below; link:changelog-details/[per-date detailed notes] live under `changelog-details/` (named `YYYY-MM-DD - title`).

The format is based on https://keepachangelog.com/en/1.1.0/[Keep a Changelog],
and this project adheres to https://semver.org/spec/v2.0.0.html[Semantic Versioning].

Entries below are in **chronological order** by date: prior release, then the **2026-03-04** org creation and repository move, then the next release.
