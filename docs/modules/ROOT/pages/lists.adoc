= Lists
:navtitle: Lists
:description: Demonstrates ordered, unordered, and description lists.

This page showcases different list types and nesting.

== Unordered Lists

=== Basic Unordered List

* First item
* Second item
* Third item
* Fourth item

=== Nested Unordered List

* Level 1 item A
** Level 2 item A1
** Level 2 item A2
*** Level 3 item A2a
*** Level 3 item A2b
** Level 2 item A3
* Level 1 item B
** Level 2 item B1

=== List with Complex Content

* First item with a longer description that wraps to multiple lines to demonstrate how the dark theme handles list item text.

* Second item with inline formatting: *bold*, _italic_, and `code`.

* Third item with a link: https://antora.org[Antora Documentation]

* Fourth item with multiple paragraphs:
+
This is a continuation paragraph.
It belongs to the fourth item.
+
And another continuation paragraph.

== Ordered Lists

=== Basic Ordered List

. First step
. Second step
. Third step
. Fourth step

=== Nested Ordered List

. Main step one
.. Sub-step 1.1
.. Sub-step 1.2
... Sub-sub-step 1.2.1
... Sub-sub-step 1.2.2
.. Sub-step 1.3
. Main step two
.. Sub-step 2.1
. Main step three

. This starts at five

. Seven
. Eight

== Description Lists

=== Basic Description List

Term 1:: Definition for term 1.
This can span multiple lines.

Term 2:: Definition for term 2.

Term 3:: Definition for term 3.

=== Description List with Nested Content

First Term::
This is the definition of the first term.
+
It includes a continuation paragraph.

Second Term::
This term has a code block:
+
[source,javascript]
----
const example = "code in description list";
----

Third Term::
This term has a nested list:
+
* Sub-item A
* Sub-item B
* Sub-item C
