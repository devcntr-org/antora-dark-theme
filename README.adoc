= Antora Dark Theme

image:https://img.shields.io/badge/license-MIT-blue.svg[License: MIT, link=LICENSE]
image:https://img.shields.io/badge/antora-3.x-purple.svg[Antora 3.x, link=https://antora.org]

Dark mode supplemental UI for link:https://antora.org[Antora] documentation sites.

== Install

[source,bash]
----
pnpm add -D github:devcntr-org/antora-dark-theme
----

Then configure your `antora-playbook.yml`:

[source,yaml]
----
ui:
  bundle:
    url: https://gitlab.com/antora/antora-ui-default/-/jobs/artifacts/HEAD/raw/build/ui-bundle.zip?job=bundle-stable
    snapshot: true
  supplemental_files: ./node_modules/antora-dark-theme/supplemental-ui
----

== Features

* Dark mode toggle button (sun/moon icons)
* System preference detection
* Persistent preference via localStorage
* No flash of unstyled content (FOUC)
